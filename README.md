# ReactSimpleStarter

A simple youtube video player application covers most basic and advanced level of React concepts.Inspired by YouTube video player where you can browse your favorite video and watch according to your mood.

Interested in this Application?

### Getting Started

There are two methods for getting started with this repo.

#### Familiar with Git?
Checkout this repo, install dependencies, then start the gulp process with the following:

```
> git clone https://github.com/madhav1993/YouTube-Video-Player-React-App.git
> cd YouTube-Video-Player-React-App
> npm install
> npm start
```

#### Not Familiar with Git?
Click [here](https://codeload.github.com/madhav1993/YouTube-Video-Player-React-App/zip/master) download the .zip file.  Extract the contents of the zip file, then open your terminal, change to the project directory, and:

```
> npm install
> npm start
```
